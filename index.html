➕ How to Add a New Month
1
Save the new month's HTML file as
feb-2026.html
in the same GitHub folder as index.html
2
Open index.html, find MONTHS_CONFIG in the script and add a new entry for that month
3
Fill in the stats: national score, gold/green count, release date, and set status: "live"
4
Push both files to GitHub. The hub updates instantly — new month card appears automatically ✅
